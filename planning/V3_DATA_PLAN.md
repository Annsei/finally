## 一、数据源选型（决定记录）

**美股历史：yfinance**（Apache 2.0 库许可 [3-0]；免 key 走 Yahoo 公开
API，数据限个人/研究/教学用途 [3-0]）。运行时按需拉取入 SQLite，
**不打包数据进仓库/镜像**（Yahoo 条款约束 [3-0]）。
落选：Alpaca（全端点要 API key [3-0]、免费档 IEX 单源+历史仅到
2016 [3-0]）；Stooq（批量包"仅限个人使用、禁商用" [3-0]、无 A 股
[2-1]、pandas-datareader 路径仅日线 [3-0]）；FirstRateData（免费仅
1 年 1 分钟样本 [3-0]，做补充参考）。

**A 股历史：AKShare（东财系接口）**——stock_zh_a_hist 日线支持
qfq/hfq 复权、免注册免 key（官方文档引述，验证票因限额未跑完，
集成时以冒烟实测为准）；刻意避开新浪系接口（文档明示封 IP 风险）。
**备选记录：Baostock**（免注册、1990 起日线+复权因子、5 万次/天/IP、
分钟线仅近 5 年 5-60m；若 AKShare 依赖过重或接口断裂即切换）。
落选：Tushare（注册积分 token 制，不符合免费免鉴权）。

**离线/CI 样本：自造固定样本集（sample source）**——确定性生成的
非真实 OHLCV 序列（趋势+回撤+震荡三形态），随仓库分发。解决三件事：
无网环境开箱可用、E2E/CI 不打外网、绕开一切数据再分发许可问题。

## 二、生态吸收结论（价值×成本排序）

1. **真实历史日线回测**（D1）：从"合成 GBM"升级为"真实行情验证
   策略"，教学价值最大；引擎沿用现有撮合/费用/T+1 语义，仅换数据。
2. **指标扩充**（D1 首批 MACD/BOLL，后续 ATR/KDJ）：自研进
   indicators.py 声明式注册表；不整包引 pandas-ta-classic（MIT、
   224 指标、无 TA-Lib 依赖 [3-0]）——它与我们的 FIELD_SPECS DSL
   映射不干净，但**作为开发期金向量校验源**（其声明式 Strategy
   分组模式 [3-0] 也佐证了"数据即配置"路线正确）。
3. **A 股实时喂价**（D2，AKShare spot 轮询源，对齐 Massive 源模式）。
4. **竞技场玩法**（D2 候选：限时私房赛/管理员事件剧本，源自 Bloom/
   StockMart 模式——首轮研究引述，票未跑完）。
5. **仅作参考不引入**：vectorbt（Commons Clause 禁售条款 [3-0]，
   教学用可但引入无必要）、backtrader/RQalpha/zvt（架构参考 [3-0]）。
