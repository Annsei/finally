## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SSE server → browser | Untrusted-shaped JSON arrives over EventSource onmessage |
| EventSource resource → page lifetime | Long-lived connection must be released on unmount |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-XSS | Tampering | usePriceStream onmessage | mitigate | JSON.parse only (no eval, no dangerouslySetInnerHTML); rendered values pass through React JSX escaping downstream |
| T-03-PP | Tampering | JSON.parse of SSE event.data | mitigate | Parse in try/catch; consumers access only known PriceUpdate keys via typed selectors; malformed payloads silently dropped |
| T-03-DoS | Denial of Service (client) | unclosed EventSource | mitigate | useEffect cleanup always calls es.close(); empty deps prevents duplicate connections |
| T-03-SSR | Information Disclosure / build failure | EventSource at module/render scope | mitigate | EventSource referenced only inside useEffect (Pitfall 3) — never evaluated during static pre-render |
</threat_model>
