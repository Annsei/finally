<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SSE store → DOM (price cells, sparkline) | Streamed values rendered into the watchlist |
| /api/watchlist → panel | JSON ticker list rendered as rows |
| chart instances → page lifetime | Each sparkline allocates a canvas that must be released |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-W-XSS | Tampering | WatchlistRow rendering of price/change%/ticker | mitigate | All values rendered as React text nodes (JSX escaping); never dangerouslySetInnerHTML with SSE data |
| T-03-W-PP | Tampering | priceUpdate fields accessed in row/sparkline | mitigate | Access only typed PriceUpdate keys; optional chaining + `—` fallback on missing fields |
| T-03-W-Leak | Denial of Service (client) | per-row Lightweight Charts canvas instances | mitigate | SparklineChart cleanup calls chart.remove() on unmount (Pitfall 4); refs nulled |
| T-03-W-SSR | Build failure | charts/EventSource during static pre-render | mitigate | createChart only inside useEffect; usePriceStream's EventSource inside useEffect — no module/render-scope browser APIs |
</threat_model>
