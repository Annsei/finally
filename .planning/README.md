# Historical GSD planning snapshot

This directory records the original V1 implementation phases and verification
artifacts from June 2026. Its STATE/ROADMAP/PROJECT/REQUIREMENTS files were not
extended for the completed platform, CN and V2 milestones and therefore are not
the current project status.

Use these sources in order:

1. `planning/CURRENT.md` for the supported product and architecture;
2. `planning/AUDIT_REMEDIATION_PLAN.md` for active hardening work;
3. `planning/OPERATIONS.md`, `planning/SECURITY.md` and `planning/API.md` for
   current runtime contracts;
4. this directory only for historical rationale and phase evidence.

Do not infer that a feature is pending solely because an old validation map or
requirement row here still says `pending`.
